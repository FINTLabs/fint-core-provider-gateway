# FINT Core Provider Gateway

FINT Core Provider Gateway er en Java/Kotlin-basert applikasjon som fungerer som en gateway mellom FINT og adaptere som leverer masterdata.

## Hvordan kjøre prosjektet lokalt

1. Klon prosjektet fra GitHub.
2. Åpne prosjektet i din foretrukne IDE (f.eks. IntelliJ IDEA).
3. Bruk Java 25 lokalt.
4. Kjør med active profile = 'local'
5. Kjør docker-compose.yaml, for eksempel med kommandolinjen 'docker-compose up -d'
6. Kjør applikasjonen

## Testing

Prosjektet bruker JUnit for enhetstesting. Kjør `./gradlew test` for å kjøre testene.
